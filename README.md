# Calculator (Java)

A simple calculator application developed in Java using a graphical user interface. The program performs basic arithmetic operations and handles user interactions through event-driven programming.

## Overview

* Built as a GUI-based calculator using Java
* Uses event handling for user input processing
* Designed to perform basic arithmetic operations
* Developed using BlueJ environment

## Features

* Arithmetic Operations

  * Addition
  * Subtraction
  * Multiplication
  * Division

* User Interface

  * Button-based input system
  * Display screen for showing results
  * Interactive layout for ease of use

* Event Handling

  * Processes button clicks using action events
  * Updates display dynamically based on input
  * Structured handling via actionPerformed method

## Technologies Used

* Java
* AWT/Swing (GUI Components)
* Event Handling (ActionListener)
* BlueJ IDE
